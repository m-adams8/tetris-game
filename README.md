# Modern Tetris Game

A visually stunning implementation of the classic Tetris game, featuring a dynamic rainbow gradient background and falling Tetris pieces animation. Built with vanilla JavaScript, HTML5, and CSS3.

![Tetris Game Screenshot](screenshot.png)

## Features

### Gameplay
- Classic Tetris mechanics with modern visuals
- Smooth piece movement and rotation
- Ghost piece preview
- Next piece display
- Score tracking and levels
- High score system
- Pause/Resume functionality

### Visual Effects
- Dynamic rainbow gradient background
- Animated falling Tetris pieces in the background
- Interactive title with hover effects
- Modern, minimalist UI design
- Retro-style font (Press Start 2P)

### Controls
- **←/→** : Move piece left/right
- **↑** : Rotate piece
- **↓** : Soft drop
- **Space** : Hard drop
- **P** : Pause/Resume game
- **Enter** : Start game

## Technical Details

### File Structure
- `index.html` - Main game structure and layout
- `tetrisGame.js` - Core game logic and mechanics
- `tetrisBackground.js` - Background animation system
- `tetrisStyles.css` - Game styling and animations

### Technologies Used
- Vanilla JavaScript (ES6+)
- HTML5
- CSS3 (Animations, Grid, Flexbox)
- Google Fonts (Press Start 2P)

### Performance Features
- Optimized background animations
- Efficient piece movement calculations
- Smooth rendering with requestAnimationFrame
- Event-driven architecture

## Installation

1. Clone the repository:
```bash
git clone https://github.com/m-adams8/tetris-game.git
```

2. Open `index.html` in a modern web browser

No additional dependencies or build steps required!
